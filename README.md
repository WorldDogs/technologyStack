# 思考

* 关于面试的一些思考：面试真的是很片面的，通过段时间的大量面试我发现，如果你的知识结构符合面试官的胃口就很容易获得好成绩，而如果关注重点不同频繁问到你的知识盲区，是无力回天的，所以要不断丰富构建自己的知识体系，唯有全面融会贯通才能立于不败之地。
